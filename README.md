### 🎛️ **Administración de Sistemas Windows 🏢**

✅ Administración de **Active Directory, GPOs, DNS, DHCP**.

✅ Virtualización con **Hyper-V y VMware**.

### 🌐 **Redes y Seguridad Informática**
